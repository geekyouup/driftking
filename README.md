## Drift King
Simple top down drifting game, vibe coded with the prompt:
`a fun top down drift car driving simulator with multiple tracks, the drifting should be realistic for a rear wheel drive car and take inputs for acceleration as well as steering`

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
